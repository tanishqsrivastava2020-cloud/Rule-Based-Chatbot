# Rule-Based Chatbot

## Description
This project is a simple chatbot that responds to user input based on predefined rules and keywords.

## Features
- Responds to greetings
- Answers basic questions
- Handles unknown inputs
- Exit command to stop chatbot

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run:
   python main.py

## Example
You: hi  
Bot: Hello!

You: how are you  
Bot: I'm fine!

You: exit  
Bot: Goodbye!
